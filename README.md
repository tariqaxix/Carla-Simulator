# CARLA Autonomous Vehicle AI

Autonomous driving agent for CARLA 0.9.13.

## Features
- A* route planning via `GlobalRoutePlanner`
- Obstacle avoidance (LiDAR + actor queries)
- Traffic light compliance and speed limit following

## Requirements
- CARLA 0.9.13, Python 3.7, `numpy`, `networkx`

## Run
```bash
py -3.7 ai_test.py -m 2

```
